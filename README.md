# void hub | OMNI-PRISM
### Professional Game Asset Marketplace & Unreal Engine Archive

## 🚀 Overview
ARIZEN is a high-performance marketplace platform designed for Unreal Engine developers. It features a "Hyper-Stitch" UI aesthetic, dynamic user profiles, and a comprehensive technical archive.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Database:** MySQL with SQLAlchemy ORM
- **Frontend:** Tailwind CSS, FontAwesome, Google Fonts
- **Authentication:** Flask-Login with secure password hashing

## 💎 Key Features
- **Team Ecosystem:** Integrated profiles for Rohit, Harvar, Dinesh, Bhargav, and more.
- **Dynamic Profiles:** Unique levels, bios, and social IDs (Discord/Insta) for every user.
- **Unreal Archives:** Extensive documentation on Skeletons, Chaos Vehicles, AI Turrets, and more.
- **Secure Gateway:** Professional checkout interface with UPI and Card support.
- **Auto-Seed:** System automatically populates core team data on first initialization.

## 📦 Installation
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Configure your MySQL credentials in `app.py`.
4. Run `python app.py` to initialize and start the server.
